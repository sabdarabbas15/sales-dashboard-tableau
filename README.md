## 📊 Project Overview
This project analyzes retail sales data to identify trends, profitability issues, and business opportunities.

## 🛠 Tools Used
- Tableau
- Excel

## 🔍 Key Insights
- West region has highest sales but lower profit margins
- Some sub-categories are loss-making
- Sales peak during Nov–Dec (seasonal trend)
- Top customers contribute major revenue

## 💡 Business Recommendations
- Optimize discount strategies
- Focus on high-profit products
- Improve performance in low-profit regions
